# DJ54B API

## Setup

```sh
$ bundle install
```

## Run

```sh
$ puma
```

## Usage

```sh
$ curl http://localhost:9292/play
$ curl http://localhost:9292/pause
$ curl http://localhost:9292/next
$ curl http://localhost:9292/up
$ curl http://localhost:9292/down
$ curl http://localhost:9292/info
```
